# Zero-CPU v0.1 Release Checklist

This checklist is for preparing the current project as a portfolio-ready v0.1 milestone.

## Required checks

```text
[ ] cmake --build build
[ ] scripts\test_all.bat
[ ] zero_cli run-os examples\bio_os
[ ] zero_studio.exe opens successfully
[ ] Studio Run BIO-OS button works
[ ] Studio trace shows Debug ASCII: BU
[ ] Studio system panel shows Timer Interrupt Count: 1
[ ] README explains CLI BIO-OS demo
[ ] README explains Studio BIO-OS demo
[ ] generated artifacts are ignored by Git
```

## Expected CLI demo

```bat
.\build\Debug\zero_cli.exe run-os examples\bio_os
```

Expected signs:

```text
ASCII view:
BU
Timer interrupt count = 1
Timer enabled = false
```

## Expected Studio demo

```bat
.\build\Debug\zero_studio.exe
```

Click:

```text
Run BIO-OS
```

Expected signs:

```text
Zero-CPU Studio v0.8
BIO-OS execution finished successfully.
Debug ASCII: BU
Timer Interrupt Count: 1
Timer Enabled: false
```

## v0.1 definition

Zero-CPU v0.1 means:

```text
- custom ISA works
- assembler works
- .zbin format works
- binary loader works
- virtual CPU execution works
- stack/CALL/RET works
- ALU/FLAGS works
- MMIO works
- DebugOutputDevice works
- TimerDevice works
- interrupt controller works
- INT/IRET works
- INT 80 syscall convention works
- mini-kernel style syscall handler works
- BIO-OS demo runs from CLI
- BIO-OS demo runs from Studio
```

## Suggested tag

After all checks pass:

```bat
git tag v0.1
git push origin v0.1
```
